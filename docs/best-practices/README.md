## 💡 最佳实践要点

### 分析前

```
✓ 明确问题和目标
✓ 选择合适的框架
✓ 准备必要的数据
✓ 预留充足的时间
```

### 分析中

```
✓ 遵循MECE原则
✓ 用数据支撑结论
✓ 深入到下一层
✓ 保持客观中立
✓ 使用检查清单
```

### 分析后

```
✓ 提炼有价值的洞见
✓ 制定可执行的方案
✓ 结论先行表达
✓ 请他人review
✓ 持续迭代优化
```
