```
我的问题是...

├─ 要不要进入这个市场/行业？
│  └→ PEST分析 + 波特五力 + SWOT
│
├─ 如何实现增长？
│  └→ 安索夫矩阵 + BCG矩阵 + 产品生命周期
│
├─ 如何降低成本/提升效率？
│  └→ 价值链分析 + 波特五力
│
├─ 如何与竞争对手竞争？
│  └→ 波特五力 + 战略钟 + 战略群组分析
│
├─ 如何设计商业模式？
│  └→ 商业模式画布 + 价值链分析
│
├─ 如何评估多个业务/项目？
│  └→ BCG矩阵 + GE矩阵 + 财务战略矩阵
```

## 🔍 按问题类型选择

### 1. 市场进入决策

**问题**：要不要进入这个新市场/行业？

**推荐框架组合**：
```
第1步：PEST分析（宏观环境是否有利？）
  ↓
第2步：波特五力（行业是否有吸引力？）
  ↓
第3步：SWOT（我们是否有优势？）
  ↓
第4步：商业模式画布（如何设计商业模式？）
```

**预计耗时**：4-6小时

**案例参考**：[在线教育SaaS进入职业教育市场](../../examples/mini-cases/market-entry-saas-education.md)

---

### 2. 增长战略制定

**问题**：如何实现业务增长？

**推荐框架组合**：
```
第1步：安索夫矩阵（增长方向是什么？）
  ↓
第2步：BCG矩阵（哪些业务值得投资？）
  ↓
第3步：产品生命周期（产品处于什么阶段？）
  ↓
第4步：价值链分析（如何建立竞争优势？）
```

**预计耗时**：5-8小时

---

### 3. 成本优化/效率提升

**问题**：如何降低成本或提升运营效率？

**推荐框架组合**：
```
第1步：价值链分析（哪些环节有优化空间？）
  ↓
第2步：波特五力（行业压力在哪里？）
  ↓
第3步：SWOT（我们的优劣势是什么？）
```

**预计耗时**：3-5小时

---

### 4. 竞争策略制定

**问题**：如何应对竞争对手？

**推荐框架组合**：
```
第1步：波特五力（竞争态势如何？）
  ↓
第2步：战略群组分析（竞争对手如何定位？）
  ↓
第3步：战略钟（我们应该选择什么定位？）
  ↓
第4步：价值链分析（对手优势在哪里？）
```

**预计耗时**：4-6小时

**案例参考**：[地方外卖平台应对美团饿了么](../../examples/mini-cases/competitive-analysis-food-delivery.md)

---

### 5. 商业模式创新

**问题**：如何创新商业模式？

**推荐框架组合**：
```
第1步：商业模式画布（现有模式是什么？）
  ↓
第2步：价值链分析（哪些环节可以创新？）
  ↓
第3步：安索夫矩阵（增长机会在哪里？）
  ↓
第4步：SWOT（创新的可行性如何？）
```

**预计耗时**：4-6小时

---

### 6. 业务组合优化

**问题**：多个业务如何分配资源？

**推荐框架组合**：
```
第1步：BCG矩阵（业务组合现状如何？）
  ↓
第2步：GE矩阵（更细致的业务评估）
  ↓
第3步：财务战略矩阵（财务策略如何？）
```

**预计耗时**：3-5小时

## ⭐ 五大最常用框架（新手必学）

### 1. SWOT分析 ⭐⭐⭐⭐⭐

**一句话**：分析优势、劣势、机会、威胁

**适用场景**：
- 快速诊断（30分钟）
- 战略规划的起点
- 任何需要内外部环境分析的场景

**优点**：简单易用，全面
**缺点**：容易流于表面

**何时使用**：
- ✅ 需要快速梳理现状
- ✅ 战略规划的第一步
- ❌ 不适合深度分析

[详细文档](./swot-analysis.md)

---

### 2. 波特五力模型 ⭐⭐⭐⭐⭐

**一句话**：分析行业竞争强度和盈利潜力

**适用场景**：
- 行业选择
- 市场进入决策
- 竞争战略制定

**优点**：系统性强，逻辑清晰
**缺点**：主要适用于传统行业

**何时使用**：
- ✅ 评估行业吸引力
- ✅ 制定竞争策略
- ❌ 快速变化的新兴行业

[详细文档](./porter-five-forces.md)

---

### 3. 商业模式画布 ⭐⭐⭐⭐⭐

**一句话**：一页纸描述整个商业模式

**适用场景**：
- 创业公司
- 商业模式创新
- 竞争对手分析

**优点**：直观、系统、易沟通
**缺点**：缺少财务细节

**何时使用**：
- ✅ 设计新商业模式
- ✅ 评估商业模式可行性
- ✅ 向投资人展示

[详细文档](./business-model-canvas.md)

---

### 4. BCG矩阵 ⭐⭐⭐⭐⭐

**一句话**：评估业务组合，指导资源配置

**适用场景**：
- 多业务公司
- 投资决策
- 资源分配

**优点**：简单直观，易于决策
**缺点**：过于简化

**何时使用**：
- ✅ 多业务组合管理
- ✅ 投资优先级排序
- ❌ 单一业务公司

[详细文档](./bcg-matrix.md)

---

### 5. 价值链分析 ⭐⭐⭐⭐⭐

**一句话**：拆解价值创造过程，找到竞争优势来源

**适用场景**：
- 成本优化
- 差异化战略
- 竞争对手分析

**优点**：深入、系统
**缺点**：耗时较长

**何时使用**：
- ✅ 寻找成本优化空间
- ✅ 建立差异化优势
- ✅ 分析竞争对手

[详细文档](./value-chain-analysis.md)

## 🎓 按学习阶段选择

### 新手（第1-2周）

**必学框架**（按顺序）：
1. **SWOT分析**（30分钟掌握）
   - 最简单、最通用
   - 建立结构化思维

2. **波特五力**（40分钟掌握）
   - 行业分析必备
   - 理解竞争本质

3. **商业模式画布**（60分钟掌握）
   - 创业必备
   - 系统化思考商业逻辑

**学习建议**：
- 每个框架看1个案例
- 尝试分析身边的企业
- 不要贪多，先吃透3个

---

### 进阶（第3-4周）

**推荐框架**：
4. **价值链分析**（55分钟掌握）
   - 理解竞争优势来源
   - 成本分析必备

5. **PEST分析**（45分钟掌握）
   - 宏观环境分析
   - 市场进入决策必备

6. **安索夫矩阵**（50分钟掌握）
   - 增长战略制定
   - 资源配置决策

**学习建议**：
- 结合真实案例练习
- 尝试组合使用框架
- 开始做完整的战略分析

---

### 高级（第5-8周）

**深化框架**：
7. **BCG矩阵**（35分钟掌握）
8. **GE矩阵**（50分钟掌握）
9. **战略钟**（45分钟掌握）
10. **战略群组分析**（40分钟掌握）

**学习建议**：
- 选择真实项目实践
- 形成自己的分析套路
- 开始教别人（最好的学习方式）

## 💡 选择框架的3个原则

### 原则1：问题驱动，而非框架驱动

```
❌ 错误："我们做个SWOT分析吧"
   （为了分析而分析）

✅ 正确："我们要解决[问题]，哪个框架最合适？"
   （问题导向）
```

**实践**：
1. 先明确问题
2. 再选择框架
3. 不要为了用框架而用框架

---

### 原则2：组合使用，而非单一框架

```
❌ 错误：只用SWOT就做战略决策

✅ 正确：组合使用
   PEST（宏观） → 五力（行业） → SWOT（综合）
   → 安索夫（战略） → 价值链（优势）
```

**实践**：
- 没有一个框架能解决所有问题
- 通常需要3-5个框架组合
- 由外到内、由宏观到微观

---

### 原则3：框架是工具，洞见是目标

```
❌ 错误：填完框架就结束

✅ 正确：框架 → 洞见 → 战略 → 行动

流程：
1. 用框架收集和组织信息
2. 从数据中提炼洞见
3. 基于洞见制定战略
4. 制定具体行动计划
```

## ❓ 常见问题

### Q1：我应该全部学会吗？

**A**：不必。建议：
- **新手**：先掌握SWOT、五力、商业模式画布（3个）
- **进阶**：再学价值链、PEST、安索夫矩阵（+3个）
- **高级**：根据工作需要选择性学习其他框架

---

### Q2：一个项目需要用几个框架？

**A**：视复杂度而定：
- **快速咨询**：1-2个框架（30分钟）
- **中度分析**：3-4个框架（2-3小时）
- **深度分析**：5-7个框架（1-2天）
- **完整战略规划**：8-10个框架（1-2周）

---

### Q3：框架之间有冲突怎么办？

**A**：框架不会冲突，它们从不同角度分析：
- **PEST**：宏观环境（外部）
- **五力**：行业竞争（外部）
- **SWOT**：内外部综合
- **价值链**：内部能力
- **商业模式画布**：整体设计

组合使用，相互补充。

---

### Q4：如何判断框架用对了？

**A**：检查清单：
- [ ] 框架帮助我回答了核心问题
- [ ] 得出了有价值的洞见（而非填空）
- [ ] 洞见指导了具体行动
- [ ] 分析逻辑清晰、有说服力
- [ ] 利益相关方认可分析结果
