# 框架组合方案库

> **标准化的分析套路**：针对常见场景，提供经过验证的框架组合方案

---

## 为什么需要框架组合

### 单一框架的局限

```
问题：
├─ 单一框架视角有限
├─ 难以全面分析问题
├─ 容易遗漏关键因素
└─ 分析深度不够

解决：
└─ 组合使用多个框架
    ├─ 从不同角度分析
    ├─ 相互补充和验证
    ├─ 形成完整分析链条
    └─ 提升分析质量
```

## 方案1：完整战略规划（5-7个框架）

**适用场景**：制定企业年度战略、3-5年战略规划

**时间要求**：1-2周

**框架组合**：

```
PEST分析 → 波特五力 → SWOT → 价值链 → BCG矩阵 → 安索夫矩阵 → 平衡计分卡

阶段1：外部环境分析（2-3天）
├─ PEST分析：宏观环境扫描
│   • 政治、经济、社会、技术因素
│   • 识别机会和威胁
│   • 预测未来趋势
│
└─ 波特五力：行业竞争分析
    • 行业吸引力评估
    • 竞争强度分析
    • 盈利潜力判断

阶段2：内外部综合分析（2-3天）
├─ SWOT分析：整合内外部
│   • 优势、劣势、机会、威胁
│   • SO、WO、ST、WT策略
│   • 战略方向初步确定
│
└─ 价值链分析：内部能力评估
    • 识别价值创造环节
    • 找到竞争优势来源
    • 确定核心能力

阶段3：战略制定（3-4天）
├─ BCG矩阵：业务组合分析
│   • 评估各业务单元
│   • 资源配置决策
│   • 投资优先级排序
│
└─ 安索夫矩阵：增长战略
    • 市场渗透、产品开发
    • 市场开发、多元化
    • 增长路径选择

阶段4：战略执行（2-3天）
└─ 平衡计分卡：战略落地
    • 财务、客户、内部流程、学习成长
    • 战略地图
    • KPI体系

预期输出：
├─ 战略分析报告（50-100页）
├─ 战略规划文档
├─ 行动计划
└─ KPI体系
```

## 方案2：快速市场分析（3个框架）

**适用场景**：市场进入决策、快速机会评估

**时间要求**：2-3天

**框架组合**：

```
PEST分析 → 波特五力 → SWOT

Day 1：宏观环境（PEST）
├─ 政治：政策支持？监管风险？
├─ 经济：市场规模？增长率？
├─ 社会：需求趋势？消费习惯？
└─ 技术：技术成熟度？颠覆风险？

Day 2：行业分析（波特五力）
├─ 供应商议价能力
├─ 买方议价能力
├─ 现有竞争者
├─ 潜在进入者
└─ 替代品威胁

Day 3：综合评估（SWOT）
├─ 机会（O）：来自PEST和五力
├─ 威胁（T）：来自PEST和五力
├─ 优势（S）：我们的资源能力
├─ 劣势（W）：我们的短板
└─ 决策：进入 vs 不进入

预期输出：
└─ 市场分析报告（10-20页）
```

## 方案3：组织诊断（3个框架）

**适用场景**：组织效能评估、组织变革准备

**时间要求**：1-2周

**框架组合**：

```
麦肯锡7S → 价值链分析 → SWOT

阶段1：组织分析（7S模型）
├─ 共享价值观：文化基因
├─ 战略：方向是否清晰？
├─ 结构：是否支撑战略？
├─ 系统：流程是否高效？
├─ 技能：能力是否匹配？
├─ 人员：人才是否到位？
└─ 风格：文化是否健康？

阶段2：能力分析（价值链）
├─ 主要活动：
│   • 生产运营
│   • 市场营销
│   • 销售服务
│
└─ 支持活动：
    • 人力资源
    • 技术研发
    • 采购管理
    • 基础设施

阶段3：综合诊断（SWOT）
├─ 优势：7S和价值链中的强项
├─ 劣势：7S和价值链中的弱项
├─ 机会：外部环境机会
├─ 威胁：外部环境威胁
└─ 改进建议

预期输出：
├─ 组织诊断报告
├─ 改进建议
└─ 变革路线图
```

## 方案4：创新战略（3个框架）

**适用场景**：商业模式创新、产品创新、战略转型

**时间要求**：1-2周

**框架组合**：

```
蓝海战略 → 商业模式画布 → VRIO

阶段1：价值创新（蓝海战略）
├─ 绘制战略画布
│   • 当前竞争格局
│   • 竞争要素分析
│
└─ 四步动作框架
    • 消除：哪些要素应该消除？
    • 减少：哪些要素应该减少？
    • 提升：哪些要素应该提升？
    • 创造：哪些要素应该创造？

阶段2：商业模式设计（商业模式画布）
├─ 价值主张：创造什么价值？
├─ 客户细分：服务谁？
├─ 渠道通路：如何触达？
├─ 客户关系：如何维护？
├─ 收入来源：如何赚钱？
├─ 关键资源：需要什么资源？
├─ 关键业务：做什么？
├─ 关键合作：与谁合作？
└─ 成本结构：成本是什么？

阶段3：竞争优势评估（VRIO）
├─ V（价值）：有价值吗？
├─ R（稀缺）：稀缺吗？
├─ I（难模仿）：难以模仿吗？
└─ O（组织化）：能充分利用吗？

预期输出：
├─ 创新战略方案
├─ 商业模式设计
└─ 竞争优势分析
```

## 方案5：竞争分析（4个框架）

**适用场景**：竞争对手分析、竞争策略制定

**时间要求**：1周

**框架组合**：

```
波特五力 → 战略群组分析 → 价值链分析 → VRIO

阶段1：行业竞争态势（波特五力）
└─ 识别竞争压力来源

阶段2：竞争定位（战略群组）
├─ 识别战略群组
├─ 绘制战略群组图
└─ 找到战略空间

阶段3：竞争优势来源（价值链）
├─ 对比自己与竞争对手的价值链
├─ 识别差异化环节
└─ 找到成本优势来源

阶段4：持久优势评估（VRIO）
└─ 评估哪些优势是持久的

预期输出：
├─ 竞争分析报告
├─ 竞争策略建议
└─ 差异化方案
```

## 方案6：并购评估（4个框架）

**适用场景**：并购尽职调查、整合规划

**时间要求**：2-3周

**框架组合**：

```
波特五力 → 麦肯锡7S → VRIO → SWOT

阶段1：行业吸引力（波特五力）
└─ 目标公司所在行业是否有吸引力？

阶段2：组织评估（7S）
├─ 价值观是否兼容？
├─ 战略是否协同？
├─ 组织如何整合？
└─ 文化是否冲突？

阶段3：资源评估（VRIO）
├─ 目标公司有哪些核心资源？
├─ 这些资源是否有价值？
├─ 是否稀缺？难以模仿？
└─ 整合后能否充分利用？

阶段4：综合评估（SWOT）
├─ 并购后的优势
├─ 并购后的劣势
├─ 协同效应（机会）
└─ 整合风险（威胁）

预期输出：
├─ 尽职调查报告
├─ 估值建议
└─ 整合方案
```

## 方案7：产品战略（3个框架）

**适用场景**：产品规划、产品组合优化

**时间要求**：1周

**框架组合**：

```
产品生命周期 → BCG矩阵 → 安索夫矩阵

阶段1：产品现状分析（生命周期）
├─ 各产品处于什么阶段？
│   • 导入期、成长期、成熟期、衰退期
├─ 不同阶段的策略？
└─ 产品组合是否健康？

阶段2：资源配置（BCG矩阵）
├─ 明星产品：重点投入
├─ 现金牛：收割利润
├─ 问题产品：选择性投入
└─ 瘦狗产品：退出或维持

阶段3：增长策略（安索夫矩阵）
├─ 市场渗透：现有产品+现有市场
├─ 产品开发：新产品+现有市场
├─ 市场开发：现有产品+新市场
└─ 多元化：新产品+新市场

预期输出：
├─ 产品战略规划
├─ 资源配置方案
└─ 增长路线图
```

## 如何选择框架组合

### 选择原则

```
1. 问题导向
   ├─ 根据要解决的问题选择
   ├─ 不要为了用框架而用框架
   └─ 框架是工具，不是目的

2. 由外到内
   ├─ 先分析外部环境（PEST、五力）
   ├─ 再分析内部能力（价值链、7S）
   └─ 最后综合（SWOT）

3. 由宏观到微观
   ├─ 先看宏观（PEST）
   ├─ 再看行业（五力）
   └─ 最后看企业（价值链）

4. 相互验证
   ├─ 不同框架的结论应该一致
   ├─ 如果矛盾，需要重新分析
   └─ 多角度验证提高可信度

5. 控制数量
   ├─ 快速分析：2-3个框架
   ├─ 中度分析：3-5个框架
   ├─ 深度分析：5-7个框架
   └─ 不要超过7个（过犹不及）
```
