# ✅ 框架使用检查清单

> **确保分析质量**：使用检查清单，避免常见错误，提升分析专业度

---

## 📋 通用检查清单（适用所有框架）

### 分析前检查

```
□ 问题定义清晰吗？
  • 具体、可衡量、有边界、有时限

□ 选择的框架合适吗？
  • 框架能回答核心问题
  • 框架与问题匹配

□ 数据准备充分吗？
  • 有足够的数据支撑
  • 数据来源可靠
  • 数据时效性好

□ 时间资源足够吗？
  • 预留足够的分析时间
  • 不要赶工应付
```

---

### 分析中检查

```
□ 是否遵循MECE原则？
  • 各要素相互独立
  • 完全穷尽所有可能

□ 是否有数据支撑？
  • 每个结论都有依据
  • 区分事实和推测
  • 标注数据来源

□ 是否深入到下一层？
  • 避免循环论证
  • 挖掘根本原因
  • 不停留在表面

□ 是否保持客观？
  • 避免主观臆断
  • 考虑多种可能
  • 不预设立场
```

---

### 分析后检查

```
□ 是否得出了洞见？
  • 不只是罗列信息
  • 有深层次的发现
  • 洞见有行为导向性

□ 是否可执行？
  • 建议具体明确
  • 有行动计划
  • 有责任人和时间表

□ 是否沟通清晰？
  • 结论先行
  • 逻辑清晰
  • 易于理解

□ 是否经过验证？
  • 与他人讨论
  • 数据验证
  • 逻辑验证
```

## 🎯 SWOT分析检查清单

### 优势（Strengths）

```
□ 是否基于事实而非主观判断？
  ❌ "我们团队很强"
  ✅ "我们有10年行业经验，核心团队稳定"

□ 是否具体可衡量？
  ❌ "产品质量好"
  ✅ "产品合格率99.5%，客户满意度90%"

□ 是否与竞争对手对比？
  ❌ "我们技术先进"
  ✅ "我们的技术比竞争对手领先1-2年"

□ 是否识别了核心优势？
  • 3-5个最重要的优势
  • 不要罗列过多
```

---

### 劣势（Weaknesses）

```
□ 是否诚实面对？
  • 不回避真实问题
  • 不美化劣势

□ 是否具体？
  ❌ "品牌知名度不够"
  ✅ "品牌认知度仅15%，竞争对手达60%"

□ 是否可改进？
  • 识别可以改进的劣势
  • 制定改进计划
```

---

### 机会（Opportunities）

```
□ 是否来自外部环境？
  • 不是内部能力
  • 是外部趋势和变化

□ 是否具体可行？
  ❌ "市场很大"
  ✅ "目标市场300亿，年增长25%，我们可切入50亿细分市场"

□ 是否有时间窗口？
  • 机会的时效性
  • 窗口期多长
```

---

### 威胁（Threats）

```
□ 是否真实存在？
  • 不是假想的威胁
  • 有证据支撑

□ 是否评估了影响？
  • 威胁的概率
  • 威胁的影响程度
  • 应对措施

□ 是否考虑了时间？
  • 威胁何时到来
  • 有多少准备时间
```

---

### SWOT策略

```
□ 是否制定了四类策略？
  • SO策略（优势+机会）
  • WO策略（劣势+机会）
  • ST策略（优势+威胁）
  • WT策略（劣势+威胁）

□ 策略是否可执行？
  • 具体的行动
  • 明确的责任人
  • 清晰的时间表

□ 策略是否有优先级？
  • 不是所有策略都要做
  • 识别最重要的2-3个
```

## 🏢 波特五力检查清单

### 五种力量分析

```
□ 是否分析了所有五种力量？
  • 供应商议价能力
  • 买方议价能力
  • 现有竞争者竞争
  • 潜在进入者威胁
  • 替代品威胁

□ 每种力量是否有具体分析？
  • 不只是打分
  • 有事实和数据支撑
  • 说明强弱的原因

□ 是否评估了力量的变化趋势？
  • 当前状态
  • 未来趋势（增强/减弱）
  • 影响因素
```

---

### 行业吸引力评估

```
□ 是否综合评估了行业吸引力？
  • 不只看单一力量
  • 综合五种力量
  • 得出明确结论（高/中/低）

□ 是否识别了关键成功因素？
  • 在这个行业成功需要什么
  • 我们是否具备
  • 如何获得

□ 是否制定了竞争战略？
  • 基于五力分析
  • 具体可执行
  • 有差异化
```

## 🎨 商业模式画布检查清单

### 九个模块

```
□ 是否填写了所有九个模块？
  • 价值主张
  • 客户细分
  • 渠道通路
  • 客户关系
  • 收入来源
  • 关键资源
  • 关键业务
  • 关键合作
  • 成本结构

□ 每个模块是否具体？
  ❌ "提供优质服务"
  ✅ "提供24小时在线客服+专属客户经理"

□ 模块之间是否一致？
  • 价值主张与客户细分匹配
  • 关键资源支撑关键业务
  • 收入覆盖成本
```

---

### 商业模式验证

```
□ 是否回答了关键问题？
  • 为谁创造价值？（客户细分）
  • 创造什么价值？（价值主张）
  • 如何交付价值？（渠道+关系）
  • 如何赚钱？（收入来源）
  • 成本是什么？（成本结构）

□ 商业模式是否可行？
  • 收入>成本（可盈利）
  • 资源可获得（可实现）
  • 客户愿意付费（有需求）

□ 是否有差异化？
  • 与竞争对手不同
  • 有独特价值
  • 难以模仿
```

## 📈 BCG矩阵检查清单

### 业务分类

```
□ 是否正确分类了所有业务？
  • 明星业务（高增长+高份额）
  • 现金牛业务（低增长+高份额）
  • 问题业务（高增长+低份额）
  • 瘦狗业务（低增长+低份额）

□ 分类标准是否清晰？
  • 市场增长率的分界线（如10%）
  • 相对市场份额的分界线（如1.0）
  • 标准合理且一致

□ 数据是否准确？
  • 市场增长率数据来源
  • 市场份额计算方法
  • 竞争对手数据
```

---

### 资源配置策略

```
□ 是否制定了每类业务的策略？
  • 明星：增长策略，加大投入
  • 现金牛：收割策略，维持投入
  • 问题：选择策略，选择性投入
  • 瘦狗：退出策略，减少投入

□ 资源配置是否合理？
  • 投资优先级明确
  • 资源总量可承受
  • 考虑了业务协同

□ 是否考虑了业务组合平衡？
  • 有现金牛支撑明星
  • 不过度依赖单一业务
  • 有未来增长点
```

## 🏗️ 麦肯锡7S检查清单

### 七要素评估

```
□ 是否评估了所有七个要素？
  • 共享价值观
  • 战略
  • 结构
  • 系统
  • 技能
  • 人员
  • 风格

□ 每个要素是否有具体描述？
  • 不只是打分
  • 有事实和案例
  • 说明现状和问题

□ 是否区分了硬性和软性要素？
  • 硬性（战略、结构、系统）
  • 软性（价值观、技能、人员、风格）
  • 理解改变难度不同
```

---

### 一致性检查

```
□ 七个要素是否协调一致？
  • 战略与结构一致
  • 系统支撑战略
  • 技能匹配战略
  • 人员认同价值观
  • 风格强化价值观

□ 是否识别了不一致的地方？
  • 哪些要素不匹配
  • 不一致的影响
  • 如何调整

□ 是否制定了改进计划？
  • 优先改进哪些要素
  • 改进的顺序
  • 预期效果
```

## 💎 VRIO框架检查清单

### 四维度评估

```
□ 是否评估了所有四个维度？
  • V（价值）：能利用机会或抵御威胁吗？
  • R（稀缺）：有多少竞争对手拥有？
  • I（难模仿）：复制成本有多高？
  • O（组织化）：企业能充分利用吗？

□ 每个维度是否有具体分析？
  • 不只是是/否
  • 有事实和数据
  • 说明判断理由

□ 是否得出了正确的结论？
  • V=否 → 竞争劣势
  • V=是,R=否 → 竞争均势
  • V=是,R=是,I=否 → 暂时优势
  • V=是,R=是,I=是,O=否 → 未实现优势
  • V=是,R=是,I=是,O=是 → 持久优势
```

---

### 战略建议

```
□ 是否针对不同类型资源制定了策略？
  • 持久优势 → 持续投入保护
  • 暂时优势 → 快速利用
  • 未实现优势 → 组织化
  • 竞争均势 → 保持或强化
  • 竞争劣势 → 剥离或改进

□ 是否识别了核心竞争力？
  • 哪些资源是真正的核心
  • 为什么难以模仿
  • 如何持续保持

□ 是否制定了资源配置计划？
  • 投资优先级
  • 资源分配
  • 预期回报
```

## 💡 使用建议

### 何时使用检查清单

**分析前**：
- 选择框架时，查看对应检查清单
- 了解框架使用的关键要点
- 准备必要的数据和信息

**分析中**：
- 边分析边对照检查清单
- 确保不遗漏关键要素
- 保持分析质量

**分析后**：
- 用检查清单自查
- 发现问题及时修正
- 提升分析质量

---

### 如何使用检查清单

**1. 打印或复制**：
- 打印出来，实战时对照使用
- 或复制到你的分析文档中

**2. 逐项检查**：
- 不要跳过任何一项
- 发现问题及时修正
- 全部勾选后再提交

**3. 持续改进**：
- 记录常犯的错误
- 重点关注这些项
- 形成个人检查清单
