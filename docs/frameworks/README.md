# 📊 经典分析框架库

本目录收录了战略分析中最常用的经典框架，每个框架都提供了详细的理论解释、实战案例和使用模板。

---

## 📚 框架分类

### 🌍 宏观环境分析

**[PEST分析](./pest-analysis.md)** ⭐⭐⭐⭐⭐
- **用途**：分析政治、经济、社会、技术等宏观环境因素
- **适用场景**：市场进入决策、战略规划、环境扫描
- **核心价值**：识别宏观机会和威胁
- **阅读时长**：45分钟

**[波特钻石模型](./porter-diamond-model.md)** ⭐⭐⭐⭐
- **用途**：分析国家或地区的产业竞争优势
- **适用场景**：产业政策制定、国际化战略、区域发展规划
- **核心价值**：理解为什么某些国家的特定产业具有全球竞争力
- **阅读时长**：50分钟

---

### 🏢 行业竞争分析

**[波特五力模型](./porter-five-forces.md)** ⭐⭐⭐⭐⭐
- **用途**：分析行业竞争强度和盈利潜力
- **适用场景**：行业选择、竞争战略制定、进入壁垒评估
- **核心价值**：理解行业结构和竞争力量
- **阅读时长**：40分钟

**[战略群组分析](./strategic-group-analysis.md)** ⭐⭐⭐⭐
- **用途**：识别行业内的竞争群组和战略空间
- **适用场景**：竞争定位、差异化战略、市场细分
- **核心价值**：找到行业中的战略空白和机会
- **阅读时长**：40分钟

**[战略钟](./strategy-clock.md)** ⭐⭐⭐⭐⭐
- **用途**：选择竞争战略定位（成本领先、差异化、混合战略等）
- **适用场景**：战略定位选择、竞争策略制定
- **核心价值**：明确企业在价格-价值维度的定位
- **阅读时长**：45分钟

---

### 📈 增长战略

**[安索夫矩阵](./ansoff-matrix.md)** ⭐⭐⭐⭐⭐
- **用途**：制定企业增长战略（市场渗透、产品开发、市场开发、多元化）
- **适用场景**：增长战略规划、资源配置决策
- **核心价值**：系统梳理增长选项，评估风险
- **阅读时长**：50分钟

**[产品生命周期](./product-life-cycle.md)** ⭐⭐⭐⭐⭐
- **用途**：分析产品所处阶段，制定相应营销和战略策略
- **适用场景**：产品战略、营销策略、投资决策
- **核心价值**：根据生命周期阶段优化资源配置
- **阅读时长**：60分钟

---

### 💼 业务组合分析

**[BCG矩阵](./bcg-matrix.md)** ⭐⭐⭐⭐⭐
- **用途**：评估业务组合，指导资源配置
- **适用场景**：多业务公司的战略规划、投资决策
- **核心价值**：识别明星、现金牛、问题、瘦狗业务
- **阅读时长**：35分钟

**[GE矩阵](./ge-matrix.md)** ⭐⭐⭐⭐
- **用途**：多因素业务组合分析（行业吸引力 × 业务实力）
- **适用场景**：复杂业务组合管理、战略投资决策
- **核心价值**：比BCG矩阵更全面的业务评估
- **阅读时长**：50分钟

**[财务战略矩阵](./financial-strategy-matrix.md)** ⭐⭐⭐⭐
- **用途**：基于价值创造和现金流状况制定财务战略
- **适用场景**：资本配置、融资决策、分红政策
- **核心价值**：指导投资、融资、分红等财务决策
- **阅读时长**：55分钟

---

### 🔗 价值创造分析

**[价值链分析](./value-chain-analysis.md)** ⭐⭐⭐⭐⭐
- **用途**：拆解企业价值创造过程，识别竞争优势来源
- **适用场景**：成本优化、差异化战略、竞争对手分析
- **核心价值**：找到竞争优势的真正来源
- **阅读时长**：55分钟

---

### 🎯 商业模式设计

**[商业模式画布](./business-model-canvas.md)** ⭐⭐⭐⭐⭐
- **用途**：系统化设计、评估、创新商业模式
- **适用场景**：创业公司、商业模式创新、竞争对手分析
- **核心价值**：一页纸描述整个商业模式
- **阅读时长**：60分钟

---

### 🔍 综合分析

**[SWOT分析](./swot-analysis.md)** ⭐⭐⭐⭐⭐
- **用途**：分析优势、劣势、机会、威胁
- **适用场景**：战略规划的起点、快速诊断
- **核心价值**：内外部环境综合分析
- **阅读时长**：30分钟

---

### 🏗️ 组织分析

**[麦肯锡7S模型](./mckinsey-7s.md)** ⭐⭐⭐⭐⭐ 🆕
- **用途**：系统性分析组织效能，评估七大要素的协调性
- **适用场景**：组织诊断、组织变革、并购整合
- **核心价值**：识别组织问题，指导变革路径
- **阅读时长**：55分钟

---

### 💎 资源能力分析

**[VRIO框架](./vrio-framework.md)** ⭐⭐⭐⭐⭐ 🆕
- **用途**：评估资源和能力，识别持久竞争优势来源
- **适用场景**：竞争优势分析、资源配置决策、并购评估
- **核心价值**：区分暂时优势和持久优势
- **阅读时长**：45分钟

---

### 🌊 创新战略

**[蓝海战略](./blue-ocean-strategy.md)** ⭐⭐⭐⭐⭐ 🆕
- **用途**：跳出红海竞争，开创无竞争的蓝海市场
- **适用场景**：商业模式创新、产品创新、战略转型
- **核心价值**：价值创新，同时实现差异化和低成本
- **阅读时长**：40分钟

---

### 🎯 战略执行与管理

**[平衡计分卡](./balanced-scorecard.md)** ⭐⭐⭐⭐⭐
- **用途**：从财务、客户、内部流程、学习成长四维度管理战略绩效
- **适用场景**：战略落地、绩效管理、目标分解
- **核心价值**：将战略转化为可衡量的指标体系
- **阅读时长**：60分钟

**[利益相关者分析](./stakeholder-analysis.md)** ⭐⭐⭐⭐
- **用途**：识别、评估、管理关键利益相关者
- **适用场景**：项目管理、变革管理、危机应对
- **核心价值**：理解各方诉求，制定沟通策略
- **阅读时长**：45分钟

**[ESG绩效衡量](./esg-performance.md)** ⭐⭐⭐⭐
- **用途**：评估企业在环境、社会、治理方面的绩效
- **适用场景**：可持续发展战略、ESG报告、投资决策
- **核心价值**：平衡财务目标与可持续发展
- **阅读时长**：50分钟

## 🗺️ 框架选择指南

### 按分析目的选择

```
1. 我想进入一个新市场/行业
   ├─ 第1步：PEST分析（宏观环境是否有利？）
   ├─ 第2步：波特五力（行业是否有吸引力？）
   ├─ 第3步：SWOT（我们是否有优势？）
   └─ 第4步：商业模式画布（如何设计商业模式？）

2. 我想制定增长战略
   ├─ 第1步：安索夫矩阵（增长方向是什么？）
   ├─ 第2步：BCG矩阵（哪些业务值得投资？）
   └─ 第3步：价值链分析（如何建立竞争优势？）

3. 我想降低成本/提升差异化
   ├─ 第1步：价值链分析（哪些环节有优化空间？）
   ├─ 第2步：波特五力（行业压力在哪里？）
   └─ 第3步：SWOT（我们的优劣势是什么？）

4. 我想创新商业模式
   ├─ 第1步：商业模式画布（现有模式是什么？）
   ├─ 第2步：价值链分析（哪些环节可以创新？）
   └─ 第3步：安索夫矩阵（增长机会在哪里？）

5. 我想分析竞争对手
   ├─ 第1步：波特五力（行业竞争态势如何？）
   ├─ 第2步：价值链分析（对手优势在哪里？）
   ├─ 第3步：商业模式画布（对手模式是什么？）
   └─ 第4步：SWOT（我们如何应对？）
```

---

### 按企业阶段选择

| 企业阶段 | 优先使用的框架 | 理由 |
|---------|--------------|------|
| **初创期** | 商业模式画布 → SWOT → 安索夫矩阵（市场渗透） | 聚焦验证商业模式，快速市场渗透 |
| **成长期** | 安索夫矩阵 → 价值链分析 → 波特五力 | 制定增长战略，建立竞争优势 |
| **成熟期** | BCG矩阵 → 价值链分析 → PEST分析 | 优化业务组合，寻找新机会 |
| **转型期** | 商业模式画布 → SWOT → PEST分析 | 重新设计商业模式，应对环境变化 |

---

### 按分析深度选择

```
快速诊断（30分钟）：
└─ SWOT分析（快速梳理内外部环境）

中度分析（2-3小时）：
├─ PEST分析（宏观环境）
├─ 波特五力（行业竞争）
└─ BCG矩阵（业务组合）

深度分析（1-2天）：
├─ 价值链分析（价值创造）
├─ 商业模式画布（商业模式）
└─ 安索夫矩阵（增长战略）

完整战略规划（1-2周）：
以上所有框架组合使用
```

## 🔗 框架之间的关系

```
           宏观环境（外部）
                ↓
           PEST分析 ←───────┐
                ↓          │
           行业分析         │
                ↓          │
         波特五力模型       │
                ↓          │
         内外部综合 ────────┘
                ↓
           SWOT分析
                ↓
         战略制定（内部）
         ├─ 增长战略：安索夫矩阵
         ├─ 业务组合：BCG矩阵
         ├─ 价值创造：价值链分析
         └─ 商业模式：商业模式画布
```

**关键逻辑**：
1. **由外到内**：先分析外部环境（PEST、五力），再看内部能力（价值链、SWOT）
2. **由宏观到微观**：先看宏观环境，再看行业，最后看企业
3. **框架组合**：没有一个框架能解决所有问题，需要组合使用

## 💡 使用技巧

### 技巧1：不要孤立使用单一框架

```
❌ 错误：只用SWOT分析就做战略决策

✅ 正确：组合使用
├─ PEST（宏观机会威胁）
├─ 五力（行业吸引力）
├─ SWOT（内外部综合）
├─ 安索夫（增长方向）
└─ 价值链（竞争优势）

结果：全面、深入、可执行的战略
```

---

### 技巧2：根据问题选择框架，而非框架找问题

```
❌ 错误："我们做个SWOT分析吧"（为了分析而分析）

✅ 正确："我们要解决[问题]，哪个框架最合适？"
例：
├─ 问题："要不要进入这个行业？"
│   └─ 框架：PEST + 波特五力
├─ 问题："如何实现增长？"
│   └─ 框架：安索夫矩阵 + BCG矩阵
└─ 问题："如何降低成本？"
    └─ 框架：价值链分析
```

---

### 技巧3：框架是工具，不是答案

```
❌ 错误：填完框架就结束了

✅ 正确：框架 → 洞见 → 战略 → 行动

流程：
1. 用框架收集和组织信息
2. 从数据中提炼洞见
3. 基于洞见制定战略
4. 制定具体行动计划
5. 执行并持续迭代
```

---

### 技巧4：定期更新分析

```
环境在变化,分析也要更新：

PEST分析：每年更新（宏观环境变化慢）
波特五力：每年或重大事件时更新
SWOT：每半年更新
价值链：每年更新（内部变化相对慢）
商业模式画布：业务调整时更新
BCG矩阵：每季度更新（业务动态变化）
安索夫矩阵：战略调整时更新
```

## 🎓 实战练习

### 练习1：选择一家公司，完成以下分析

```
推荐公司：
├─ 特斯拉（新能源汽车）
├─ 字节跳动（互联网）
├─ 星巴克（连锁零售）
└─ 华为（科技制造）

分析任务：
1. PEST分析（宏观环境）
2. 波特五力（行业竞争）
3. SWOT（内外部环境）
4. 价值链分析（竞争优势来源）
5. 商业模式画布（商业模式）
6. 安索夫矩阵（增长战略）

输出：
一份完整的战略分析报告（10-20页）
```

---

### 练习2：对比两家竞争对手

```
推荐对比：
├─ 可口可乐 vs 百事可乐
├─ 麦当劳 vs 肯德基
├─ 苹果 vs 三星
├─ 阿里巴巴 vs 京东
└─ 美团 vs 饿了么

分析任务：
1. 用价值链分析对比两家的价值创造过程
2. 用商业模式画布对比两家的商业模式差异
3. 用SWOT分析各自的优劣势
4. 得出结论：谁的战略更优？为什么？

输出：
竞争对手对比报告（5-10页）
```

---

### 练习3：为自己设计商业模式（个人版）

```
如果你是"产品"：

用商业模式画布分析：
├─ 价值主张：你能为雇主/客户解决什么问题？
├─ 客户细分：你的目标雇主/客户是谁？
├─ 客户关系：如何建立和维护关系？
├─ 渠道：如何让客户知道你？（领英、作品集等）
├─ 收入来源：工资、咨询费、培训费等
├─ 关键资源：你的技能、知识、网络
├─ 关键业务：你每天做什么创造价值？
├─ 关键合作：谁能帮助你？（导师、合作伙伴）
└─ 成本结构：你的"投入"是什么？（时间、学习成本）

输出：
个人商业模式画布（1页）
```

## ❓ 常见问题

**Q1：这些框架是否过时？**
A：经典框架的底层逻辑不会过时。虽然诞生于几十年前，但PEST、五力、价值链等框架的思维方式仍然适用。关键是灵活运用，结合新的商业环境。

**Q2：我应该全部学会吗？**
A：不必。建议先掌握SWOT、五力、商业模式画布这3个最通用的框架。然后根据你的工作需要，选择性深入学习其他框架。

**Q3：框架会限制创新思维吗？**
A：不会。框架是帮助你结构化思考的工具，而非限制。好的分析师会灵活使用框架，甚至改进框架。关键是"用框架，不被框架用"。

**Q4：实际工作中，真的有人用这些框架吗？**
A：是的。顶尖咨询公司（麦肯锡、BCG、贝恩）、大企业战略部门、MBA课程都在使用这些框架。它们是战略分析的"标准语言"。

**Q5：如何避免"为了分析而分析"？**
A：始终牢记分析的目的。不要为了完成框架而填空，而是为了解决具体问题而使用框架。问题驱动，而非框架驱动。
