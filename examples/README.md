## 📝 示例格式规范

每个示例应包含：

```
example-name/
├── README.md           # 示例说明和学习目标
├── problem.md          # 问题描述
├── solution.md         # 解决方案和步骤
├── assets/             # 图片、图表等资源
└── exercise.md         # 练习题（可选）
```

### README 应包含
- 🎯 学习目标
- ⏱️ 预计时间
- 📋 前置知识
- 🔑 关键要点
- 💡 延伸思考

## 🤝 贡献示例

我们需要：
- 清晰简洁的教学示例
- 容易理解的入门案例
- 有趣的练习题
- 实用的工具演示

### 好示例的标准
- ✅ 目标明确，聚焦单一知识点
- ✅ 步骤清晰，易于跟随
- ✅ 包含实际可操作的练习
- ✅ 提供答案或参考解法
- ✅ 时间控制在合理范围
