## 版本说明

### 版本格式
- **主版本号(Major)**：进行不兼容的 API 修改时
- **次版本号(Minor)**：添加向下兼容的功能时
- **修订号(Patch)**：进行向下兼容的问题修正时

### 变更类型
- `新增(Added)` - 新功能
- `变更(Changed)` - 现有功能的变更
- `弃用(Deprecated)` - 即将移除的功能
- `移除(Removed)` - 已移除的功能
- `修复(Fixed)` - Bug 修复
- `安全(Security)` - 安全相关的修复
